## Setup
1. Clone the repo: git clone https://github.com/username/repo.git
2. Install dependencies: npm install
3. Run Docker container: docker-compose up
4. Deploy to AWS using GitHub Actions
## Usage
The app automatically builds and deploys on push to main branch.
